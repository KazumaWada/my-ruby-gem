# 🚀 quick start

```bash
gem install kazuma_wada --user-install
```

```bash
irb -r kazuma_wada
```
irb を起動したら、以下を実行してください：<br>
irb（Interactive Ruby): Rubyのコードを1行ずつ入力してすぐ実行できる対話環境 
```bash
irb(main):001:0> KazumaWada.profile
```
